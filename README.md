# Vue-Backstage-Manage

這是一個使用Vue完成的項目，它能幫助使用者管理後台資訊

## 項目展示




## User Story 

1. 用戶列表: 管理使用者
2. 權限管理: 管理使用者的權限
3. 商品管理: 管理商品的增刪修改以及分類還有標籤
4. 訂單管理: 管理訂單進度
5. 數據統計: 可視化消費者主要來源

## 使用到的技術

* Vue CLI3
* Vue Router 
* axios串接資料
* element-ui
* Echarts
